AVK - AWS SDK + Vue.js + Koa

You can manage your Amazon S3 bucket taking advantage of the Service Worker's ability of intercepting requests in order to cache the responses to improve performance. You can also drag and drop files and folders to be uploaded to the bucket you happen to be at.

This is a Progressive Web App created using the following libraries/technologies:

  • MaterializeCSS
  • AWS SDK
  • Koa
  • Vue.js
  • Service Worker
  • PWA
About the baby icon:

child_care

Update the bucket you are in. The idea is to fetch the newest version (therefore a baby's face), rather than the old one (from cache).
